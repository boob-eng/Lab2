# Задание 1
### 1.Постановка задачи
- Условие задачи
    >Задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.
- Необходимо выявить закономерность от введённых `a0` и `n` и реализовать алгоритм для построения последовательности для любых входных значений
- Чтобы выявить закономерность будем вводить разные входные значения `a0` и `n` и анализировать полученные последовательности
- При вводе разных значений `a0` видно каждый следующий член последовательности равен сумме цифр предыдущего, умноженную на 2

2. Подпрограмма:
    Чтобы найти сумму цифр создаю переменную, в которую буду последовательно прибавлять каждую цифру числа, при этом убирая ее дальше
4. Выполнение цикла и вывод:    
   Программа в цикле умножает сумму цифр начального числа на 2 и выводит его. Также ставит запятые после каждого числа, кроме последнего
   Следует отметить, что с отрицательными a0 последовательность действует одинаково
### 5. Код программы
```java
import java.io.PrintStream;
import java.util.Scanner;
public class Dis
{
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static int Sum(int a)
    {
        int s=0;
        while (a>0){
            s+=a%10;
            a/=10;
        }
        return s;
    }
    public static void main(String[] args)
    {
        int a0=Math.abs(in.nextInt());
        int n=in.nextInt();
        for (int i=0;i<n;i++){
            a0=2*Sum(a0);
            out.print(a0);
            if (i<n-1){
                out.print(", ");
            }
        }
    }
}
````

# Задание 3
### 1.Постановка задачи

> Микрорайон с начальной численностью населения P0 развивается по годам: каждый год к текущему населению прибавляется прирост, равный целой части выражения P * g / 100, после чего учитывается миграция m. 2. Цикл и вывод:    
    Цикл работает пока 'флаг' истиннен. Сначала сохраняем кошличество населения в отдельную переменную, увеличиваем население согласно условию задачи. Дальше проверяем, превысило ли население 'Cap'
   или нет. Если население уменьшилось, то останавливаемся,иначе продолжаем. После заверпшения цикла выводим год и население.

### 5.Код программы
```java
import java.io.PrintStream;
import java.util.Scanner;
public class n3
{
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args)
    {
        int p =in.nextInt();
        int g=in.nextInt();
        int m=in.nextInt();
        int Cap=in.nextInt();
        int y=0;
        boolean boo =true;
        while (boo){
            int prevP= p;
            int increase=(p*g)/100;
            p=p+increase+m;
            if (p>Cap){
                p=Cap;
                y++;
                boo=false;
            }
            if (p<=prevP){
                p=prevP;
                boo =false;
            }else{
                y++;
            }
        }
        out.println(y);
        out.println(p);
    }
}
```

# Задание 4
### 1.Постановка задачи
- Условие задачи
>Дана последовательность из n элементов. Известно, что все ее элементы находятся в диапазоне от 10 до 100 (включая границы). Определить, есть ли такие элементы, которые встречаются в последовательности ровно столько раз, каково их значение. Если такие элементы есть – вывести их значения, если их нет – вывести "NO".

- Данная задача предполагает анализ количества чисел в последовательности, работу с массивами. 
  
### 2. Входные и выходные данные

#### Данные на вход

Согласно условию задачи на вход подаётся целое число `n` являющееся значением длины массива, а также элементы самой последовательности. 

| Число |  Тип   |Нижняя граница|Верхняя граница |
|-------|--------|--------------|----------------|
| n     |Целое   |      0       |2<sup>31</sup>-1|
| {arr} |Целые   |      10      |100             |

#### Данные на выход
Для выходных данных понадобится булиновая переменная и целочисленная переменная, которая работает как индекс в одном из циклов, так и значение ответа.

| Число |Тип    |Нижняя граница|Верхняя граница|
|-------|-------|--------------|---------------|
| j     | Целое |     10       |100            |

### 3. Выбор структуры данных
Для хранения длины массива будем использовать переменную `n` типа данных `int`, сам массив также будет целочисленным. Для хранения количества данного числа в последованности будем использовать целочисленный массив []dss.

| Число     | Название переменной |Тип переменной|
|-----------|---------------------|--------------|
| n         | n                   |     int      |
| []arr     | []arr               |     int      |

Результат есть число

| Число | Название переменной | Тип переменной |
|-------|---------------------|----------------|

### 4. Алгоритм

#### Алгоритм выполнения программы

1. Ввод данных:    
   Программа считывает натуральное число `n`, создаёт массив этой длины и с помощью цикла заполняет его значениями, введёнными с клавиатуры. Также создается массив длинны 100 (чтобы легче работать с индексами), чтобы хранить значения количества элементов в последовательности. Создается переменная 'флаг'(изначально false), чтобы отслеживать, есть ли нужные нам числа.
2. Цикл нахождения нужного числа:
    Задаём начальное значение индекса 10 с границей до 100 включительно, сравниваем значение индекса с значением массива dss данного индекса. Так мы получаем число, которое встречалась в последовательности столько раз, сколько равно данное число. Выводится на экран это число, 'флаг' становится истинным. Если 'флаг' так и не стал истинной,то нужных нам чисел так и не нашлось-выводим "NO". 

### 5.Код программы
```java
import java.io.PrintStream;
import java.util.Scanner;
public class n4
{
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;
    public static void main(String[] args)
    {
        int n=in.nextInt();
        if (n<10){
            out.println("NO");
        }else {
            int[] arr = new int[n];
            for (int i = 0; i < n; i++) {
                arr[i] = in.nextInt();
            }
            int[] dss = new int[101];
            for (int i = 0; i < n; i++) {
                dss[arr[i]]++;
            }
            boolean boo = false;
            for (int j = 10; j <= 100; j++) {
                if (dss[j] == j) {
                    out.print(j + " ");
                    boo = true;
                }
            }
            if (!boo) {
                out.println("NO");
            }
        }
    }
}
```
